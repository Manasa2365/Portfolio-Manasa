Web Development Portfolio using HTML, CSS & JavaScript

Introduction:

As part of the web development training session, I learned the core web technologies: HTML, CSS, and JavaScript. To practically apply these skills, I created my own portfolio website to showcase my projects, skills, and achievements in a visually engaging and interactive format.

Objectives:

  • To understand the structure and semantics of HTML.
  • To explore styling and layout design with CSS including gradients, animations, and custom cursors.
  • To add interactivity and responsiveness using JavaScript.

Features Implemented:

  • Responsive design compatible with both desktop and mobile screens.
  • Custom cursor trail effect using JavaScript.
  • Animated headings and glowing UI elements.
  • Sections for About, Skills, Projects, Achievements, and Contact.
  • Smooth scrolling and hover transitions for better UX.

Skills Gained:

  • Semantic HTML5 structure and tags
  • Advanced CSS styling, flexbox, gradients, and shadows
  • DOM manipulation and event handling in JavaScript
  • Design thinking and UI/UX layout planning

Conclusion:

This assignment not only strengthened my foundational knowledge in web development but also gave me the confidence to build real-world UI experiences. The project reflects my learning progress and ability to integrate design with functionality.

In the future, I plan to enhance this portfolio with backend integration, project filtering, and theme switching features.
